# Animation System

## Philosophy

Animation exists to guide attention.

Never animate for decoration only.

Motion should feel like turning pages in a book.

---

# Library

Anime.js

# Motion Style

Elegant

Slow

Natural

Editorial

Premium

---

# Motion Principles

Fade before move.

Move before scale.

Never bounce excessively.

Never shake elements.

Keep animations subtle.

# Performance

60 FPS target.

Use transform and opacity.

Avoid animating width, height, left, top.

Lazy animate below-the-fold sections.

Respect prefers-reduced-motion.
