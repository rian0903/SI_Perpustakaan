# Layout System

## Design Philosophy

The website follows a cinematic storytelling layout.

Each section is a chapter.

Each chapter has a beginning, focus, and transition.

# Content Rhythm

Every section alternates between:

Text Left / Image Right

↓

Image Left / Text Right

to create visual rhythm and prevent monotony.

---

# Experience Flow

The user journey should feel like reading chapters of a premium book:

1. Curiosity
2. Welcome
3. Discovery
4. Engagement
5. Trust
6. Exploration
7. Action
8. Connection

Every layout decision should reinforce this narrative while keeping the interface clean, modern, and highly readable.
